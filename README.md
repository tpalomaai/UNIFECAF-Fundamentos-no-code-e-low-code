Sobre o Projeto
Este projeto consiste em um portal de gerenciamento de tarefas e comunicação interna, desenvolvido utilizando conceitos de No-Code/Low-Code com a plataforma Bubble.

A aplicação permite que usuários:

Criem e gerenciem tarefas

Atribuam responsáveis

Adicionem comentários

Recebam notificações de atualizações

O objetivo é facilitar a organização das atividades e a comunicação entre membros de uma equipe.

Estrutura do Sistema
Principais entidades do banco de dados:

User – usuários do sistema

Task – tarefas criadas pelos usuários

Comment – comentários nas tarefas

Notification – notificações geradas por interações

Principais Páginas
Dashboard (Index) – painel principal com tarefas e notificações

Login – autenticação de usuários

Cadastro – criação de novas contas

Tarefa – visualização e comentários de uma tarefa específica

Reset Password – redefinição de senha

404 – página de erro para rotas inexistentes
